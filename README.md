# CheZay - Restaurant Management App

CheZay is a **Restaurant Management Mobile Application** developed as a university project.  
The app is designed to improve restaurant operations and customer experience by allowing users to browse menu items, place orders, reserve tables, and track orders easily.

---

## Project Overview

The main goal of **CheZay** is to provide a digital solution for restaurant management.  
It helps both **customers** and **restaurant staff** by making restaurant services more organized, efficient, and user-friendly.

---

## Main Features

### Customer Features
- User Registration and Login
- Browse Restaurant Menu
- Search Food by Name or Category
- Add Items to Cart
- Place Food Orders
- Track Order Status
- View Order History
- Reserve a Table
- Manage Profile

### Admin Features
- Manage Menu Items
- View and Manage Orders
- Manage Restaurant Staff
- View Sales Overview
- Generate Reports

### Staff Features
- View Incoming Orders
- Update Order Status
- Manage Reservations
